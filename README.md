# MindMach: Interactive Onboarding Bot

An interactive, terminal-based onboarding script built with Python. Inspired by platforms like PDB (Personality Database), this project serves as the registration module for a fictional social network called MindMach, designed for psychology enthusiasts to lock in their MBTI and Enneagram profiles.

## Key Features and UX Enhancements

1- Data Sanitization: Automatically processes raw user inputs using string methods to ensure clean and standardized data entry.
2- Terminal UI Alignment: Delivers an organized user experience by displaying a perfectly aligned digital Identity Pass using precise spacing techniques.
3- Engaging User Flow: Features a multi-step sequence for user registration.

### Roadmap and Future Upgrades

As this project was developed on Day 3 of my Python learning journey, it has some limitations. In the next release, I plan to integrate:
+ Input Validation: Utilizing conditional logic and loops to reject invalid formats, such as preventing users from entering wrong Enneagram patterns.
+ Database Integration: Simulating actual storage for user profiles rather than just displaying them.
